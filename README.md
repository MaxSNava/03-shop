# Shop Teslo API

1. Clonar proyecto
```
git clone https://...
```
2. Instalar dependencias
```
npm install
```
3. Renombrar el archivo __.env.template__ a __.env__
4. Cambiar las variables de entorno de __.env__
5. Levantar la base de datos
```
docker compose up -d
```
